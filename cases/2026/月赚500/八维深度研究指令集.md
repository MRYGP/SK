## 指令一：【模式考古】国内"月入500-2000"已验证副业模式图谱

**研究目标**：从草根信息源（非媒体/非论文）挖掘当前**真实有人在做、且收入稳定在这个区间**的副业模式。重点排除"卖课型"和"概念型"，只保留"执行型"。

**信息源优先级**：

1. **生财有术/淘金之路/AI破局**：搜索"低保"、"副业打卡"、"月入500"、"学生/宝妈"关键词，筛选**连续打卡>30天、有收入截图、有具体操作步骤**的帖子
2. **小红书**：搜索"副业真实收入"、"宝妈兼职日记"、"大学生搞钱"，筛选**粉丝<1000、画面粗糙、连续更新**的素人账号
3. **闲鱼**：搜索"代"、"帮"、"做"、"整理"，筛选**成交量>50、评价真实**的服务型商品
4. **豆瓣**："副业失败小组"看失败案例（反向验证）；"副业交流小组"看低门槛帖子
5. **抖音/视频号**：搜索"副业第一天"、"这个月副业赚了"，筛选**非口播、非导师型**的真实记录

**输出要求**：

1. **模式矩阵表**：模式名称、操作平台、启动成本、时薪区间、月收入天花板、是否需要特殊技能、是否需要身份资源、是否支持利润再投资（是/否/部分）
2. **10个可验证案例**：每个案例标注信息源链接、发帖时间、操作者身份、收入描述、关键卡点
3. **模式分类**：按"无脑程度"分为S级（有手就行）、A级（需简单学习）、B级（需一定技能）
4. **复利适配度评级**：每个模式标注"是否天然支持500元再投资放大收入"（1-5星）

**证据等级**：

- L1：多平台交叉验证（如小红书+闲鱼+生财有术同时出现同类模式）
- L2：单一平台但有连续记录（>30天打卡或>10条相关帖子）
- L3：单一个体自述，无后续更新

**与主项目关联**：直接回答"我们应该主推哪3种模式"

## 指令二：【复利实证】500元再投资的6种路径ROI实测

**研究目标**：验证"利润再投资"是否真能将月收入从500放大到1500+。聚焦**500元以内、学生/宝妈可执行、有明确ROI计算**的再投资动作。

**研究路径**：

1. **设备投资**：二手iPad/降噪耳机/外接显示器对数据标注/客服/内容生产的效率提升实测（找3个真实用户访谈）
2. **流量投资**：闲鱼币推广、小红书薯条、抖音DOU+对"代做/本地服务"类帖子的曝光-成交转化率（实测或找卖家访谈）
3. **信息投资**：付费社群（99-299元档）是否能带来"企业直发包"或"高单价任务源"（访谈3个社群成员）
4. **外包投资**：将低单价任务（时薪15元）外包给他人，自己专注高单价任务（时薪30元），测算套利空间
5. **身份投资**：用家庭成员身份注册新平台账号，扩大任务池，测算边际收益
6. **工具投资**：购买付费版AI工具（ChatGPT Plus、剪映会员、Excel插件）对任务效率的真实增益

**输出要求**：

1. **6路径ROI对比表**：投资类型、成本、预期收益、回本周期、风险等级、适合人群（学生/宝妈）
2. **3个真实复利案例**：要求有"第1月收入→再投资→第2月收入"的完整记录
3. **失败案例**：500元投出去打水漂的真实记录（至少2个）
4. **最优投资组合**：基于500元本金，给出"4-3-3法则"的实证调整建议

**证据等级**：

- L1：实测数据（你自己或研究员亲自操作7天的记录）
- L2：付费社群内的实战复盘帖（有截图、有数字）
- L3：卖家/用户的口述访谈

**与主项目关联**：直接回答"工具里应该内置哪3种再投资方案"

## 指令三：【人群洞察】学生/宝妈的"微资本"行为与心理研究

**研究目标**：理解目标用户的**"再投资意愿"和"决策障碍"**。不是问"你想不想赚钱"，而是研究他们在有500元盈余时，真实的消费/投资行为。

**研究方法**：

1. **问卷设计**（样本量≥30）：
   - 如果你这个月副业赚了500元，你会怎么花？（A.存起来 B.买设备 C.买课/社群 D.犒劳自己 E.其他）
   - 你最多愿意为"提高效率"一次性投入多少钱？
   - 你更信任"AI推荐的投资方案"还是"真人社群的推荐"？
2. **深度访谈**（5人：2学生+2宝妈+1对照）：
   - 过去是否有过"赚钱后再投入"的经历？结果如何？
   - 对"花钱才能赚更多"的心理障碍是什么？
   - 什么情况下愿意把利润交给平台/工具去"自动投资"？
3. **行为观察**：
   - 观察闲鱼/小红书上的学生/宝妈卖家，他们的店铺是否有"升级"痕迹（如从手机拍摄到用支架、从纯文字到用模板）
   - 观察生财有术/豆瓣小组中，用户从"提问者"变成"分享者"的转折点

**输出要求**：

1. **用户画像升级**：从"有时间的人"升级为"有微资本决策行为的人"
2. **决策障碍清单**：阻碍利润再投资的Top 5心理因素（如"怕被骗"、"想存着应急"、"不懂怎么投"）
3. **信任锚点**：用户更愿意相信什么形式的"投资建议"？（数据图表/真人案例/AI模拟/社群共识）
4. **分层策略**：高意愿用户（愿意投）vs 低意愿用户（只想稳）的产品分层设计

**证据等级**：

- L1：问卷统计数据
- L2：访谈逐字稿+编码分析
- L3：个人经验推测

**与主项目关联**：直接回答"产品如何设计才能让用户愿意把利润再投入"

## 指令四：【国际对标】海外Micro-Entrepreneurship复利模式研究

**研究目标**：研究海外（尤其美国、东南亚）是否有成熟的"微创业复利"基础设施，验证"利润再投资"模式的可移植性。

**研究范围**：

1. **Gig Economy平台**：TaskRabbit、Fiverr、Upwork的"新手→Top Rated"路径中，是否有平台内置的"再投资引导"（如技能测试、工具推荐、流量包购买）
2. **Micro-SaaS**：海外"一人公司"如何用工具栈（Zapier、Notion、Make）实现"个人时间→系统自动化"的升级，成本是否在500元以内
3. **东南亚模式**：菲律宾/印度的"数据标注员→团队组织者"路径，如何利用低人力成本实现复利
4. **海外案例**：Starter Story、Indie Hackers上"月收入<$1000"的真实项目，分析其启动资金和再投资策略

**输出要求**：

1. **海外模式矩阵**：平台/工具名称、启动成本、复利机制、是否可移植到中国、移植障碍
2. **3个可对标案例**：要求月收入在$100-$500区间，有明确的"再投资→增长"记录
3. **基础设施差距**：海外有哪些工具/支付/信任机制是中国缺失的？（如Stripe的即时到账、PayPal的纠纷处理）
4. **反向输出**：中国有哪些海外没有的优势？（如身份资源、平台补贴、社群密度）

**证据等级**：

- L1：平台官方文档/规则页
- L2：Indie Hackers/Reddit的真实复盘帖
- L3：国内媒体的二手报道

## 指令八：【合规红线】身份资源、众包、撮合的法律风险图谱

**研究目标**：绘制"利用身份资源做副业"的**法律边界地图**。确保产品推荐的任何路径**不触碰帮信罪、非法经营、侵犯公民个人信息等红线**。

**研究范围**：

1. **身份使用边界**：
   - 用自己身份注册多个平台账号 → 是否违规？
   - 用家人身份注册并代运营 → 是否合法？
   - 招募他人身份做"账号矩阵" → 是否涉嫌侵犯个人信息？
2. **众包/众测的法律性质**：
   - 数据标注是否构成"劳动关系"？平台是否有义务缴纳社保？
   - 收入超过多少需要报税？平台代扣代缴规则？
3. **撮合服务的责任**：
   - 帮宝妈撮合"代遛娃/陪诊"服务，若出事故，平台/组织者是否承担连带责任？
   - 闲鱼上发布"代做服务"，是否属于无照经营？
4. **资金流转风险**：
   - 用户通过个人微信/支付宝收款，月流水超过多少会触发风控？
   - 平台抽成模式 vs 信息费模式，税务处理差异

**输出要求**：

1. **法律风险矩阵**：行为类型、违规概率、法律后果、规避方案、证据来源（法条/判例/律师意见）
2. **3个真实判例**：与副业/众包/身份出租相关的司法案例（裁判文书网搜索）
3. **产品合规清单**：我们的"AI微任务引擎"在功能设计、收费模式、用户协议上必须满足的**最低合规标准**
4. **保险/担保机制**：是否需要引入第三方保险或 escrow（资金托管）来降低撮合风险？

**证据等级**：

- L1：裁判文书/法条原文/律师意见书
- L2：平台用户协议中的责任条款
- L3：网络讨论的"听说"

**与主项目关联**：直接回答"哪些模式绝对不能碰，哪些需要加免责声明/保险"
