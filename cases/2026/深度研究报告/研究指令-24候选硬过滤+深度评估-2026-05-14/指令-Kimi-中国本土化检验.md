# 任务:24 个候选的中国本土化现实校验

我有 24 个候选产品方向,需要你做中国本土化检验——你的输出不评估好坏,只看**在中国能不能跑通**。

## 你的独特价值

你是 Moonshot 训练的 AI,对中国互联网生态 / 政策环境 / 用户行为 / 流量入口 / 支付链路 / 合规通道有第一手观察。这次评估的其他 AI(GPT / Claude / DeepSeek)对中国现实理解有限,你的角色是揭开他们看不到的现实障碍。

## 输出要求(对每个候选)

每个候选写 150-250 字,覆盖以下五个本土化维度:

### 维度 1:政策合规通道
- 这个方向在中国是否触及医疗 / 金融 / 教育 / 跨境 / 数据 / 内容审查等监管红线?
- 如果触及,合规通道是什么(具体到牌照 / 备案 / 接入审批)?
- 合规通道是否对 3-5 人小团队开放?

### 维度 2:流量入口现实
- 这个产品的目标用户在中国从哪里发现产品?
- 是否依赖微信 / 抖音 / 小红书 / 百度 / 知乎等主流流量?
- 这些入口是否对该品类友好(或已被限流 / 屏蔽 / 审核加重)?

### 维度 3:支付链路
- 中国用户为这类服务付费的路径是什么?
- 是个人微信/支付宝直付,还是企业对公,还是平台抽佣?
- 是否存在支付链断裂(比如医疗咨询不能直接收钱)?

### 维度 4:已有竞品现实
- 中国是否已有 3+ 个直接竞品 / 替代品?
- 这些竞品是已饱和、还是有差异化空间?
- 大厂(腾讯 / 阿里 / 字节 / 美团 / 拼多多 / 京东)是否已经在做或可能很快做?

### 维度 5:用户心智现实
- 中国目标用户是否真的会为这类服务付费(不是"应该会",是"已经在付")?
- 类似品类在中国是否有过失败案例?为什么失败?
- 中国用户对 AI 在这个领域的接受度如何?

## 重要约束

1. **不要给"理论上可行"的客套话**——只要现实
2. **不要怕给 ❌ 死路判断**,这正是你的价值
3. **特别警惕海外案例平移陷阱**:某个方向在美国跑通不代表在中国跑通。Cactus 在美国小镇个体户成立,在中国类似场景被美团/58同城/外卖系统吞掉。这种"看着像但底层基础设施不同"的陷阱要专门指出。
4. **不要做产品好坏评估**,只看本土化可行性
5. **不要参照其他候选打分**,每个独立评估
