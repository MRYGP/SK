## 中国「小作坊层(第三层)」AI 产品机会扫描

### 层级声明

**本报告仅扫描第三层(年营收 <500 万、5-30 人、无 IT、老板拍板的小作坊 / 个体工商户)**。

未扫描:

- **第一层(中大型企业)**:其 AI 采购走 IT/CIO 流程,有数字化预算,与本研究问题不同质。
- **第二层(50-500 人中小企业)**:决策链是"老板+部门负责人",有人对接 SaaS 但缺开发能力。本报告未扫第二层,**原因:它与第三层动机迥异,需单独成文,混扫会冲淡判断**。

第三层的"反 AI 创业"基本盘(读下文前必须先承认):

- 老板月付费天花板 **50-2000 元**;高于 2000 立刻被视为"骗钱"。
- WeChat 是**事实操作系统**,绕开微信意味着丧失存量入口。
- 老板对"新 APP"高度排斥,但对"现有工具突然变聪明"接受度高。
- 个体户三年存活率约 30%(国家市监总局历年小微企业公告,B 级),LTV 极短。
- 大量"工具"已被腾讯/字节/拼多多/美团免费铺出,创业者在这层做横向工具的死亡率极高。

带着上述底色,下面 6 个候选都是"已经在第三层每天发生"的真痛,**但不代表都能赚到钱**,扣分会在评分里讲清。

------

### 场景 1·小装修队 / 个体包工头 现场报价 + 增项确认

**谁、什么时刻、卡什么、土办法**:全国 200 万+ 量级小装修队 / 个体包工头(X 级,无权威统计),老板带 3-15 个工人。关键时刻:业主走完毛坯房,问"全包多少",老板拍胸脯报数,后续每周冒出 2-5 项增项扯皮。中消协《2024 年全国消协组织受理投诉情况分析》披露房屋装修类投诉 21216 件,位列服务大类第九(A 级)。土办法:微信文字 + Excel + 老板脑子。

**已有替代 + AI 能补的缝**:行业充斥盗版 CAD/SU、口头报价;"蜡笔装修报价计算器"、装小蜜等小程序面向**业主反查**装修队,**没有给装修队的"AI 增项预判 + 业主签字闭环"工具**。AI 缝:业主发房屋照 → 自动出潜在增项清单;中途增项时生成"为什么增、增多少、业主签字"的微信流程。证据等级 B。

**嵌入入口**

| 入口存在吗?                                                  | 入口可用吗?                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 存在但极度碎片化。老板用个人微信、相册、Excel。58 同城、土巴兔、好工长等平台有 CRM 模块但渗透率低。**没有统一"装修队工作台"**。 | 难。包工头不装新 APP;唯一现实路径是**微信小程序 + 企微插件**,业主侧链接 / 老板侧服务号。直接 BD 单店成本极高,需绑定 58 / 土巴兔 / 抖音家装频道才能批量铺。 |

**两种模式**

- *现在自己干*:**不可行**。缺三个关键能力 —— 300+ 城市装修工长 BD 网络 / 58 与土巴兔渠道关系 / 业主端口碑。
- *先招募再启动*:可行。需招 ① 有 58 / 贝壳 / 土巴兔行业 BD 副总(必备,80-200 万/年);② 微信小程序 + 企微全栈;③ 一名一线工长出身的产品经理(关键,无此人产品必偏)。验证第一步:在一个城市签 30 家工长,跑通"业主-装修队-增项确认"闭环 3 个月。

**评分:5/10**。扣分:渠道极碎 + BD 周期长 + 月费 50-300 元经常付不下来。装修队老板的真痛是"招到活+收到钱",AI 工具排在 5 个紧迫问题之后。

### 场景 2·建材小店(瓷砖/地板/卫浴/橱柜)帮业主"配我家"

**谁、什么时刻**:县市级建材市场单店店主,带 2-5 个店员。关键时刻:业主拿手机里几张毛坯房照片走进店,问"哪款贴在我家厨房好看"。店主每天回答 20-50 次,**答好 = 当场成交,答差 = 走人比价**。土办法:翻店内实景样板 + 翻同行抖音 + 老板脑补。

**已有替代 + AI 能补的缝**:酷家乐、三维家有效果图工具但**面向设计师**,门槛高;美家达人、躺平设计家面向 C 端业主,**不解决店主端"30 秒搭配出图"问题**。AI 缝:店主用微信接业主照片 → 30 秒生成"业主家中实景 + 本店产品贴图 + 同色系搭配建议",回发业主。证据等级 B。

**嵌入入口**

| 入口存在吗?                                                  | 入口可用吗?                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 极强。店主天天在微信和企微和客户聊。建材品牌(东鹏、马可波罗、欧派、九牧)有经销商 SaaS。 | 取决于路径。**直接 BD 单店**:难,单店付费弱。**走品牌总部**:可行但 BD 周期 6-18 个月,且必须与既有经销商 SaaS 共存。**走美团 / 抖音本地商家工具**:抖音和美团本身在做类似功能,你是与平台争。 |

**两种模式**

- *现在自己干*:不可行,需要建材品牌或大经销商关系。
- *先招募再启动*:可行性中等。需招 ① 东鹏 / 马可波罗 / 欧派任一家总部经销商管理经验高管(决定生死);② 视觉 AI 工程师;③ 一名设计师做训练数据。验证:绑定 1 家二线建材品牌的 100 家经销商,3 个月跑出"AI 搭配图 → 转化率"对比。

**评分:6/10**。扣分:酷家乐若推"店主版"会瞬间挤压,但其过去 5 年没做,大概率认为不挣钱(B 级);天花板有限。

### 场景 3·汽修店 故障可视化解释 + 维修报价

**谁、什么时刻**:县市级独立汽修店店主,全国约 50 万+ 家(B 级,公安部机动车保有量 + 汽配联会估算),3-10 人。关键时刻:车主把车开来,师傅打开发动机舱拍几张照片,告知"变速箱要大修,8000 块",**车主立刻怀疑被宰**。土办法:打开发动机舱让车主看 + 微信发故障件照片 + 口头解释。

**已有替代 + AI 能补的缝**:智慧修车、车通云、汽配龙等汽修 SaaS 集中在配件查询和工时定额,**不解决"让车主信任方案"问题**。AI 缝:故障件照片 → 标注问题 → 自动生成"为什么必须修 + 不修后果 + 类似案例"的车主版可视报告,微信秒发。证据等级 B。

**嵌入入口**

| 入口存在吗?                                                  | 入口可用吗?                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 存在,中等强度。车通云、智慧修车等汽修 SaaS 已覆盖数十万店面(B 级)。 | 中等。理想路径是**与汽修 SaaS 共生**,做"故障解释插件";独立 BD 单店难度高于场景 1。 |

**两种模式**

- *现在自己干*:不可行,缺汽修行业渠道。
- *先招募再启动*:可行。需招 ① 车通云 / 智慧修车出身 BD 负责人;② 汽车机械 + 视觉模型工程师(稀缺);③ 5-10 个种子修理工提供标注。验证:200-500 家店跑 3 个月,看车主签字前犹豫时间从 5 分钟降到 1 分钟。

**评分:6.5/10**。扣分:汽修 SaaS 玩家未必愿分润;但汽修单店现金流好、月付 300-1000 元接受度高、场景标准化程度比装修高。

### 场景 4·单体口腔诊所 / 小型中医诊所 患教 + 复诊提醒

**谁、什么时刻**:单体口腔诊所(全国 8-10 万家,B 级,中国口腔医疗集团招股书引用)、个体中医馆。关键时刻 ① 患者治疗后医生说"两周后复诊",真实复诊率 30-50%(C 级);② 面诊报方案时,患者反复问"这么贵值不值"。土办法:医生亲自加微信发提醒 / 让护士群发 / 朋友圈发科普文 —— 几乎人人都做不到位。

**已有替代 + AI 能补的缝**:领健 LinkedCare、菲森、众择等口腔 SaaS 服务连锁居多;**单体小诊所大量用微信 + 简易记账,工具断层**。AI 缝:① 治疗后自动生成"患者听得懂的"患教图文 / 视频(基于本次治疗类型);② 个性化复诊节奏提醒;③ 患者咨询接入,医生不在岗时 AI 替答常见问题。证据等级 B。

**嵌入入口**

| 入口存在吗?                                             | 入口可用吗?                                                  |
| ------------------------------------------------------- | ------------------------------------------------------------ |
| 强。所有诊所医生用企微 / 个人微信加患者。HIS 玩家分散。 | 较好。**走企微 + 小程序**比走 HIS 集成更轻,可绕开 HIS 厂商。但**必须避开涉医广告红线** —— AI 生成的患教必须医生人工审核。 |

**两种模式**

- *现在自己干*:**有限可行**。如果创始人是口腔医生或有连锁口腔经历,从 3-5 家自营诊所做闭环。**纯互联网背景团队不可行**。
- *先招募再启动*:可行。需招 ① 有瑞尔 / 拜博 / 美维运营经验的医务总监(强需求,无此人监管直接出问题);② 法务合规(医疗广告法、患者隐私)。验证:30-50 家单体诊所跑 3 个月,看复诊率提升幅度。

**评分:6.5/10**。扣分:涉医监管不明确(AI 生成医疗内容尚无清晰判例);诊所年关闭率 15-20%,LTV 不长;月付 300-800 元 OK。

### 场景 5·县域小商家 短视频 / 直播内容自动生成 + 投流教练

**谁、什么时刻**:县城小餐饮、美甲店、私房烘焙、小服装店老板(全国个体工商户 6000-8000 万家,A 级,国家市监总局)。关键时刻:老板每天想"今天发啥视频",**80%+ 老板不会拍、不会写、不会投流**(B 级)。土办法:① 找代运营 1500-8000/月经常被坑(B 级,鸟哥笔记、知乎多份代运营报价披露);② 自己抄同行;③ 雇刚毕业小孩;④ 放弃。

**已有替代 + AI 能补的缝**:剪映 + 即创(字节自家)、千川助手、巨量小店 —— 全是字节系。腾讯有视频号助手。AI 缝看似存在 —— 给老板做"今日选题 + 文案 + 模板 + 投流预算"四件套,但**这条缝在不断被字节自己填**。证据等级 A(代运营市场价 1500-20000 月)。

**嵌入入口**

| 入口存在吗?                                | 入口可用吗?                                                  |
| ------------------------------------------ | ------------------------------------------------------------ |
| 入口非常强(抖音 / 视频号 / 巨量引擎账号)。 | **几乎不可用**。字节自家产品(即创、巨量小店、千川助手)持续吞并这个空间。任何独立工具都在与字节正面竞争。腾讯视频号有类似动作。 |

**两种模式**

- *现在自己干*:不可行。直接与字节正面竞争是创业自杀。
- *先招募再启动*:**仅在垂直行业可行**。例如专做"县域美甲店抖音 AI 教练"或"乡镇汽修直播 AI",绑定行业的工具跨过字节横向工具的护城河。

**评分:4/10**。**扣分原因:这是我最不推荐的方向**。给 4 分是因为垂直行业还有缝,但**列出来主要是为了警告,不是为了推荐**。

### 场景 6·小型来图加工厂(注塑/五金/钣金/服装小订单)询价 + 自动报价

**谁、什么时刻**:5-30 人微型加工厂老板(全国 40-60 万家,B 级,工信部中小工业企业口径)。关键时刻:每天微信收到客户图纸或样品照片,问"能不能做、多少钱、什么时候交"。老板自己半夜回,**估错就亏几千到几万**。土办法:凭经验估价 + Excel 物料表 + 手算工时。

**已有替代 + AI 能补的缝**:海智、找钢网、震坤行、1688 是撮合平台,**不解决单店报价**;阿里 ERP / 钉钉 / 飞书 / 简道云在小厂渗透不足 30%(X 级)。AI 缝:微信接图 → 识别零件类型 / 材料 / 工艺 → 调老板的物料库 + 历史报价 → 30 秒出"对客版报价单 + 内部成本表"。

**嵌入入口**

| 入口存在吗?                                                  | 入口可用吗?                                                  |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| 入口存在但分散。微信是核心。少量厂用阿里 / 钉钉,70%+ 仍纸笔 + Excel。 | 较好。**微信 / 企微插件 + 简易小程序**即可嵌入,无需替换老板任何习惯。 |

**两种模式**

- *现在自己干*:**有限可行**。如果创始人在制造业出身且认识 100 家加工厂老板,可从细分赛道(东莞注塑 / 温州五金)做闭环。
- *先招募再启动*:可行。需招 ① 细分行业 BD(必须开过厂或在加工厂干过);② 视觉 AI + CAD/STP 解析工程师(稀缺);③ 资深工艺师傅作标注教练。验证:30-50 家东莞注塑厂跑 3 个月,看"老板回复询价时间"从 4 小时降到 30 分钟、客户成交率提升。

**评分:7/10**。**这是我给分最高的场景**。理由:① 痛点真实且每日发生;② 老板月付 500-2000 元经访谈可承受;③ 数据飞轮成立 —— 每家厂的物料库 + 历史报价是独占资产;④ 工艺多样性带来天然区隔(注塑/五金/钣金/服装订单需分别建模,大厂不愿做)。扣 3 分因为:行业极细分必须单线深耕;CAD/STP 图纸解析有技术门槛;货款回收慢是制造业通病。

### 反例与放弃清单

考虑过但放弃:

1. **小餐饮店选址 / 团购定价 AI 顾问** —— 美团 / 抖音本地生活在持续推送"商家助手",独立工具撞免费工具,死亡率极高。
2. **个体律师 / 个体会计 助手** —— 群体偏小,且对 AI 错误容忍度低,法律责任难界定。
3. **菜场摊主 / 水果摊 直播 AI** —— 客单极低、操作时间极短,不存在"老板掏手机用 AI"的时刻。
4. **婚介 / 媒人 个体 AI 撮合** —— 极碎且行业灰色,KYC 监管不明。
5. **个体物流 / 货拉拉司机 调度 AI** —— 货拉拉 / 满帮 / 滴滴货运的平台层已吞掉调度。司机层只剩开车。
6. **二手房经纪 个体 AI 助手** —— 贝壳 / 我爱我家几乎垄断 SOP,逃离贝壳的人一定逃离工具。

### 评分判别力声明

本报告 6 个候选评分区间:**4-7 分**,中位数 6,标准差约 1。

- 最低 4 分(场景 5,短视频内容):与字节 / 腾讯正面相撞,本质不可行。
- 最高 7 分(场景 6,来图加工厂):数据飞轮 + 付费意愿 + 嵌入入口三个条件最干净。
- 中位 6-6.5(场景 2、3、4):有合理路径但都需要"先招募关键 BD 才能启动"。

**判别力提示**:6 个候选有 3 个落在 6-6.5,**评分接近、不可仅按分数排序**;真正的选择信号是"创始人现成有没有这条线的渠道关系"。如果你本来就在某一行(口腔医生 / 装修包工头 / 注塑二代),对应那条 +2 分;反之 -2 分。

**对非创业者读者**:默认所有评分需要 -1.5 分。创业第三层最大风险是"创始人想象的痛"与"老板真实的痛"差 50%。这份扫描不能替代一线访谈。

### 多 AI 对比时的提醒

如果把同一提示词同时丢给多个模型,大概率得到**高度一致**的答案:大家都会推荐"小餐饮"、"装修"、"小诊所"、"短视频代运营"。

**高度一致 ≠ 正确,可能是共同盲点**。模型训练数据重叠,爬的中文创投媒体重叠,导致对"显眼场景"的指认是同步的。

**矛盾处更值得深挖**:

- 我给场景 5(短视频代运营)4 分,如果别的模型给 8 分,矛盾点是"是否愿意正面撞字节" —— 这需要你自己拍板。
- 如果某模型推荐"小餐饮选址 AI"高分,矛盾点是"是否愿意撞美团 / 抖音本地生活"。
- 如果某模型推荐"个体律师 AI"高分,矛盾点是"法律责任谁担"。

**建议流程**:把多家答案的**矛盾清单**整理出来,逐条做"对赌式问答" —— 让两个模型互相反驳,你只读反驳;反驳得不出共识的地方,就是你必须亲自跑 3-5 家小作坊一线访谈才能判断的地方。任何一家模型(包括我)单独说的"高分场景",都应该被这套程序压制一次再决策。
