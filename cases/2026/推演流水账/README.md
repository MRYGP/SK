# 推演流水账目录

> 存放历次深度推演的完整过程记录
> 命名规则:`推演流水账-[主题]-[日期].md`
> 查找推演时:先看 `ops/执行状态总表.md`,再来这里看分类索引,再 list 本目录找具体文件

---

## 分类索引(按主题)

> 这里只维护分类标签,不维护逐条文件清单。**新增流水账无需更新本 README**,直接放入本目录即可。
> 想看完整文件清单,直接在本目录 `ls` 或浏览。

### 一·方法论沉淀类

- 多维综合框架起源
- 尝一口方法论
- 因子 2 因子 5 扫描

### 二·产品中国版推演类

某个海外 AI 产品在中国市场的可行性 / 落地路径推演。

- MYHAIR AI 中国版
- Yoodli 中国版
- AI 销售陪练金融垂直版

### 三·战略转向类

★★★ 重要程度高于其他两类。SK 整体战略层面的认知转向。

- 诊断 vs 匹配 vs 三方错位(2026-04-25)——SK 2026 年最重要方法论沉淀,涉及"诊断空白 / 中立角色"类话题必读

## 维护规则

- 新增流水账:直接放入本目录,文件名遵循命名规则
- **不在本 README 维护逐条文件清单**——历史教训:这种"半截清单"必然过期且无人维护
- 如果一个主题分类下流水账积累到 5+ 条,考虑该分类是否值得拆为独立子目录
