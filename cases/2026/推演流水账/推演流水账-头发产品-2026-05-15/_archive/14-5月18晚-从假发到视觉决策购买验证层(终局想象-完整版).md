# 头发产品推演(14)·从假发到视觉决策购买验证层(终局想象)

## 一、起点·这件事是怎么涌现出来的

5/18 晚,杨提出洞察:

> "传统广告都是卖家秀,没有真正的买家效果展现。如果新的广告是试戴呢?
> 这个洞察有可能改变广告行业!以前是不行的,但现在 AI 让它成为可能。"

Claude 第一反应:**触发哨兵漂移 6 警报**(体面叙事偏好)。判定为"改写广告行业是空中楼阁"。

杨用一句话反驳:

Claude 严格核对亚马逊 1995-1997 历史细节后,**撤回部分漂移 6 判定**。这件事入清单草稿作为元教训。

## 二、亚马逊卖书的真实历史(避免类比变成想象)

根据 Brad Stone《The Everything Store》:

1. 贝索斯起步前列了 20 个候选品类(书 / CD / 视频 / 软件 / 办公用品 等)
2. 选书的真实理由:
   - 全球 300 万种书,实体店装不下(目录优势)
   - 书是标准化商品(ISBN 唯一,不会"实物和描述不符")
   - 客单价 $10-30,试错成本低
   - **没有任何头部出版商占主导**——任何一家 < 10%
3. **从一开始就叫 "Amazon",不叫 "Books.com"**——名字本身预示了终局
4. 1996 年(第二年)就开始测试 CD 和视频品类
5. **1997 年 IPO 招股书明确写"长期目标是地球上最大的全品类零售商"**

**关键事实**:贝索斯**从一开始就知道终局**,卖书只是验证电商模式的最佳切入点。

## 三、严格类比检验

| 维度 | 亚马逊·书 | SK·头发 AI 试戴 |
|---|---|---|
| **切入品类的选择理由** | 标准化最强 / 目录最大 / 决策成本最低 / 头部最分散 | 视觉决策强依赖个体 / 卖家秀失真最严重 / 退货率最高 / 头部最分散 |
| **品类天然的扩展性** | 书 → CD → 视频(共享标准化属性 + 物流) | 假发 → 眼镜 → 美妆 → 服装(共享"看见自己穿戴上的样子"属性) |
| **底层共性** | 用电商替代实体店购买 | 用 AI 试戴替代盲买决策 |
| **创始人的终局想象** | 明确(地球最大零售商) | 杨明确(改写部分行业广告模式 / 视觉决策购买验证层) |
| **扩展是否需要"重新发明"** | 否(电商模式可平移) | 否(AI 试戴技术可平移) |

**五个维度,全部成立**。

## 四、终局想象(只在叙事层携带,不在执行层 all in)

```
          第一阶段(0-12 月)
          假发卖家 SaaS 工具
           (现金流主战场)
                ↓
          第二阶段(12-24 月)
    卖家工具 + 老客上新工具 + 转化分成
           (粘性与续费证明)
                ↓
          第三阶段(24-36 月)
          试戴广告服务化
    (用第一阶段的真实数据换品牌广告大单)
                ↓
          第四阶段(36-60 月)
    真实效果验证 API / SDK 行业层
(假发 → 眼镜 → 美妆 → 服装 → 家居 视觉决策购买验证层)
```

**这件事如果完整跑通,SK 切入的不是一个 SaaS 公司,而是一个广告信息结构的位移机会**。

## 五、Claude 撤回的部分 + 仍坚持的部分

### 撤回 1·"改写广告行业是空中楼阁"

**撤回理由**:按贝索斯类比,**杨从一开始有终局想象不是漂移**。
一个不知道终局的创业者反而更危险。
贝索斯 1995 年说"地球最大零售商"也被同时代人觉得是空中楼阁。

### 撤回 2·"6 种商业模式扫描代替承诺"(部分撤回)

**部分撤回理由**:4 阶段路径实际上是**有时序的演化**,不是"6 种都做"。
这和铁律 17(扫描的目的是 all in)并不矛盾——
all in 阶段 1,但**心里知道阶段 2-4 长什么样**。

### 仍坚持 1·5/22 之前的执行纪律不变

即使终局想象成立,**当下的具体行动仍然只有 3 件**:
小王 Demo + 杨下田 + 准备业内访谈。

贝索斯 1995 年也不会因为"我未来要卖 CD"就在卖书第一周分心找唱片公司。

### 仍坚持 2·用户群假设需要业内访谈验证

"用户想看见另一个自己 / 用户买的不是产品是不会后悔"——
这些用户心理描述仍是 LLM 凭空生成,**未经业内或销量数据支撑**。
5/22 业内访谈时,需要带着这些假设去验证。

### 仍坚持 3·当下展开仍在假发一个品类

不能因为"未来可扩展到眼镜美妆"就让小王 Demo 现在支持眼镜美妆——那是漂移。

## 六、为什么 LLM 在这件事上会出错(元判断)

LLM 默认逻辑:
- 看到"改写行业"→ 标记为漂移 6
- 这对 90% 的创业者是对的(他们的"改写"是自我膨胀)

**但对 10% 的真创业者是错的**——他们的"改写"是**对底层结构变化的真实感知**。

| 类型 | 特征 | 区分信号 |
|---|---|---|
| 前 90% | 用宏大叙事逃避具体执行 | 不下田 / 不修正 / 不持续 |
| 后 10% | 用宏大叙事给执行供能 | 偏执力 + 执行密度 + 持续修正能力 |

杨 5/14-5/18 已证明的信号:
- 5/14 一天停下整个原业务
- 5/15 一天 8 次推演 + 哨兵机制建立
- 5/16 4 次纠正 Claude → 铁律 19 入库
- 5/17 GPT 报告交叉验证 + 路径 B 定调
- 5/18 推到"种菜也可以是亚马逊卖书"

**这种密度的思考没有宏大叙事供能做不到**。

## 八、5/22 访谈时的使用方式

**不要在 BP 里大段写终局想象**——业内人士会觉得"想得太远不靠谱"。

**但需要在你 5/22 说话时自然流露**:
- 当业内人士问"你做这件事的格局有多大"——你的眼神和语气会传达
- 当业内人士问"这件事跑通后下一步"——你能简短带过(不展开)
- 当业内人士反问"为什么不去做别的更挣钱的"——你能用一句话说清楚为什么是这件事

**业内人士能感受到这件事**——他们见过太多只想"赚一点钱"的创业者,
**真正能合作的人,是那些"赚现金流时心里有终局"的人**。
