# 头发产品推演(04)· 行动计划与 SOP 抽取

## 第一节:5/16-5/26 行动计划

### 5/16-5/17(2 天):用户访谈 + 商家访谈

**用户访谈(5-10 个)**:
- 张已有 1 个
- 找:身边脱发的朋友 / 去过假发店没坚持的人 / 化疗康复者(如能联系到)

**提纲(每人 30 分钟)**:
1. 你脱发多久了?有没有戴过假发?
2. 第一次想戴假发的时候,最大障碍是什么?
3. 你去过假发店吗?店内试戴感觉怎么样?
4. 如果有 APP 可以在家私密地试戴各种假发,你愿意试吗?
5. 你愿意上传照片吗?如果只上传脱敏特征(脸型 / 肤色)呢?
6. 你试戴成功了愿意花多少钱买?
7. 你会推荐给身边的朋友吗?

**商家访谈(3-5 家)**:
- 在淘宝 / 拼多多 / 1688 找中等规模假发卖家(月销 500-5000 单)

**提纲(每人 30 分钟)**:
1. 现在月销多少?客单价?
2. 退货率多少?主要退货原因?
3. 如果有 AI 试戴工具帮你导流,你愿意接入吗?
4. 愿意付月费 999-2999 还是分成 10-15%?
5. 试戴 → 购买转化率你期望多少?
6. 你最希望这个工具帮你解决什么?

### 5/18-5/19(2 天):决策 + Demo 启动

- 5/18 早上:复盘所有访谈数据
- 5/18 下午:决策——继续 / 暂停 / 换方向
- 5/18 晚 - 5/19:招募 1 个前端 + 1 个数据后端(如未到位)

### 5/20-5/26(7 天):做最简陋 Demo

- 5/20-5/22:化身上传 + 脱敏 + 基础试戴(只有 5-10 个款式)
- 5/23:接入 1 家商家
- 5/24-5/25:内部测试
- 5/26:给 3-5 个真实用户看 demo

### 5/27 起:根据反馈决定走或换

- 用户上传率 > 30% + 试戴体验得分 > 7/10 → 继续
- 任一不达标 → 反思 + 改 + 再测一周
- 连续两周都不达标 → 换方向,回到候选池

## 第三节:SOP 抽取——产品方向推演七步法(SK·SOP·001)

> 这是本次推演**最大的元产出**——一个可复用的产品推演 SOP。
> 以后推演任何产品,都按这七步走。

### 第一步:画完整市场地图(Map First, Don't Pick)

- 不是按产品形态,是按**用户问题**分类
- 至少展开到第二级
- 标注每个赛道的产品形态 + 主流玩家
- 输出物:一棵两级以上的赛道树

### 第二步:四个钱维度评估

- 总盘子 + 单笔金额 + 决策频次 + 利润率
- 综合"钱密度"打分
- 输出物:钱密度排序表

### 第三步:决策摩擦评估

- 决策路径长度
- 决策周期
- 主要恐惧
- 易成交度打分
- 输出物:决策摩擦排序表

### 第四步:寄生 / 嵌入位置扫描

- 每个赛道单独评估有没有寄生位置
- 列出至少 3 个可寄生入口
- 输出物:寄生位置清单

### 第五步:四层竞争对手地图

- 直接竞品
- 替代品
- 寄生宿主
- 降维者
- 输出物:四层竞争图 + 最大降维风险点

### 第六步:多维交叉打分

- 7 个维度:钱密度 / 易成交 / 寄生 / AI 杠杆 / 焦虑强度 / 6 月现金流 / 树的潜力
- 交叉打分,选总分前三
- 输出物:多维交叉评分表 + 前三候选

### 第七步:SK 库工具全套过一遍(9 把刀)

1. 两层断层产品形态(三条件)
2. 错位母模型(五形态)
3. 认知翻译层诊断
4. 客户采用六因子(DOC-D029)
5. 第二波下场判据
6. 菜成树检验(五问)
7. AI 杠杆充分利用度
8. 失败模式扫描(failure_modes)
9. 反向 teardown(没碰的领域)

任一工具触发否决 → 回到第六步换候选
全部通过 → 进入产品形态设计

### SOP 使用纪律

- **每一步都必须输出物理产物**(表格 / 树状图 / 清单 / 评分)
- **不能跳步**(跳第一步直接打分 = 在小盒子里打分)
- **每步通过后再下一步**(第一步没做完不能做第二步)
- **每个产品方向必须做完整七步**(不能"凭感觉"选)

## 第四节:本 SOP 待验证

- 第七步的 9 把刀是否完备?需要在第 2-3 次使用后校准
- 7 个交叉打分维度是否需要调整?
- "钱密度"和"易成交"这两个判据有时冲突,如何取舍?
- 不同行业是否需要不同 SOP 变体?(医疗 vs 电商 vs 企业服务)
