# 推演流水账：ZJPT 专家对赌平台（= CAND-004）

## 第一步 · 接住模糊想法（原样记录）

- 原话：专家对赌平台 = 让专家不再卖时间/卖建议，而是「对结果负责」「对赌」，平台提供撮合 + 结算 / 担保。
- 缺失信息：谁是专家？什么"结果"？谁付费（买家 success fee / 专家抽佣 / 担保金池）？"对赌"具体是哪一种（成功才收费 / 失败赔付 / 共享上行）？平台靠什么活、靠什么防绕单？
- 语气：把"对赌+结算"当作产品本体。
- 不替它补充、不美化。

## 第二步 · 对标扫描（先抄通，找真实存在的"对结果负责"生意；中国优先）

> 铁律：必须找真实存在、能活/已死的样本，别凭框架推。以下全部是真实生意。

### 活体一 · 风险代理律师（中国 · 活 · 窄缝） —— 证据等级 A（司法部规范性文件）
- 机制：胜诉酬金（contingent fee），不过/败诉不收，成功按标的额比例提成。
- 监管：2023《意见》把上限从 30% 砍到分段累进 18%/15%/12%/9%/6%；明令禁止适用于人身损害、婚姻继承、群体性诉讼、刑事、行政、国家赔偿、社保等案型。
- 存活条件（关键）：只活在「结果客观可测（追回/减免的金额）+ 执行在律师手里 + 归因干净 + 结果在供方可控范围」的债权债务缝里。
- 隐忧：监管担心"只许胜不许败"诱发利益输送、影响司法公正 -> 对结果收费天然被"归因清洁度 + 公共利益"封顶。
- 抄通启示：能活的"成功才收费"= 执行可控 ∩ 结果可测 ∩ 归因干净，三者缺一即死。

### 活体二 · 对赌 / VAM 估值调整机制（中国 · 活但有毒） —— 证据等级 B（学术研究 + 媒体 + 判例）
- 机制：创始人对投资人承诺业绩/上市，未达成则补偿/回购 = 创始人对结果负责。
- 数据：中国 90%+ VC 投资含对赌，硅谷 <5%（张巍研究）；2024 投资机构参与回购事件 1741 起；王自如/ZEALER 对赌失败被强制执行 3383 万，年化 12% 滚到约 1 亿。
- 法律：法院对"投资方 vs 股东"对赌认可，对"投资方 vs 目标公司"对赌（动用公司资本）多否定（资本维持原则）。
- 本质（最深一刀）：中国式对赌不是健康结构，是"钱等不起+路走不通+人输不起"三重系统失灵的症状（搜狐长文）。
- 抄通启示：当"结果"不完全在承诺方控制（市场/退出/大环境），对赌 -> 债务雪球 + 诉讼机器。"对赌"这个词在中国语境背着剧毒包袱。

### 活体三 · 中公"不过包退"协议班（中国 · 正在爆雷） —— 证据等级 B（半年报 + 财经媒体）
- 机制：2010 首创"不过全退"协议班 = 机构对"考上"这个结果负责，没考上退费。曾带来滚滚营收。
- 数据：2022 起爆雷，4.82 亿学费难退；个案"1.7 万退 17 年"；公考报名 2009->2025 从 105.2 万->341.6 万，录取跑不赢报名（结果本身在恶化）。小机构直接平台外交易、收钱失联（46 名考生维权）。
- 死因（CAND-004 墓碑诊断的活体印证）：1. 机构承诺它无法控制的结果（录取率）；2. 执行（学生自己学）不在机构手里；3. 归因脏（没考上是没教好还是没学好？）；4. 重做成本高（再来一年）。从业者严叶一句话："没有任何辅导机构可以代替考生前期的基础学习。"
- 抄通启示：承诺不可控结果 + 执行不在供方手里 = 退费义务变债务黑洞 = 必爆。

### CAND-004 已列墓碑（不重复展开）
效果付费/分成代运营、币圈跟单、Upwork/Fiverr 结果保证——同一死因：执行不受供方控制 + 归因脏 + 重做成本高。

## 第三步 · 真实案例纠偏（已解锁 · 包包微信对话）—— 证据等级 A（亲历）

样本：杨向真实商家"包包"提出"基础服务费+分红的真专家代运营平台"，包包当场回"你就是那个专家""我们对赌可以""可以的，你自己研究研究，这个不难的"。我们朋友那个店也愿意。

三件冷读（别被"愿意"冲昏）：
1. 包包愿意是**信你、不是信你的能力**——你自言"我屁都不懂"，她仍愿先干后结，押的是关系抵押（CAND-005 熟人信任打底）。第一单能成 ≠ 模型成立，只证明"关系够硬时陌生人也敢先干后结"。
2. "不难、你自己研究研究"是**警报不是鼓励**——她潜意识认为运营 know-how 学得会。学得会 = 你帮她跑通那天 = 她最后一次有理由分钱给你（6-07 那个洞的 B2B 版；中公/分成代运营同死因）。
3. 真问题是**价值归谁**（6-04 那一刀）：真专家做运营拿分红、包包出店、你屁都不懂 -> 这条链里 SK 捕获什么？答不出"归我"，分红算得再漂亮也是原地打转。

纠偏结论：包包这单是"载体可下田"的入口，不是"假设已成立"的证据。真专家是否找得到/愿不愿分红进来（A03）、归因能否干净、包包尝甜头后续不续约——三件都没测过。

## 第四步 · 重新定义问题（维度跨越 / 换题——竞争对手必须变）

- 被那个洞 + 三个活体推翻的旧问题："怎么做一个让专家对结果负责的【对赌结算平台】？"
  - 那个洞：结算不是护城河（首单后绕开平台）-> 真问题不在"怎么结算/担保"。
  - 三活体共识：能活的"对结果负责"只在【结果客观可测 + 执行在供方手里 + 归因干净 + 结果可控】窄缝。
  - CAND-004 使能前提："AI 把执行搬进供方内部、重做边际成本趋零"。而"专家平台"执行还在人手里 -> 没满足使能前提 -> 默认形态踩所有墓碑死因。
- 重新定义后的问题（竞争对手变了）：
  - 旧竞争对手：支付宝担保 / escrow / 担保平台（已被那个洞 + CAND-005 escrow 墓碑证伪）。
  - 新问题："在哪个【单一窄品类】里，AI 能把执行接管进可观测/可复现的内部，使'成功才收费/达标否则退费'第一次对陌生专家在经济上可承受？"
  - 新竞争对手：那个品类里**现在按工时/月费卖的传统专业服务**。
- 这把 ZJPT 从 CAND-004 的"平台本体形态"打回 CAND-004 第 7 节的"滩头形态"。

## 第五步 · 跑历史数据（推演公式：历史数据 × 约束 × 迭代深度）

- 风险代理：收费 = 标的额 × 比例，且被监管限价 -> "对结果收费"的天花板被【归因清洁度 + 监管 + 公共利益】三重锁定。
- 对赌：2024 回购诉讼 1741 起 -> 这个结构在中国的主要产物是诉讼，不是价值创造。
- 中公：4.82 亿待退 -> 承诺不可控结果的负债规模量级。
- 频率测试（七步法第五步）：专家级"高价值结果"通常**低频**（一桩标的案 / 一笔 PE / 一次考公 / 一次手术）。低频 -> 无复购 -> 那个洞的"首单后绕开平台"在低频里更致命（根本没有第二单留在平台的机会）。
  - 反向印证那个洞：平台本体只能是"持续供给新陌生对"的供给引擎，不是结算基础设施。

## 第六步 · 增长机制

- 若 ZJPT 是"供给引擎"：增长 = 不断买新买家 + 拉新专家，CAC 高、无飞轮 -> 对标币圈跟单 / Upwork 死法（FM015 无飞轮 + FM001 流量依赖）。
- 唯一可能的飞轮：沉淀"哪个专家在哪类结果上真兑现过"的**可验 track record**（= CAND-005 的"可检成品当抵押"）。
  - 但这要求结果可检 + 可归因 -> 又回到第四步那条窄缝约束。结果越非标，飞轮越弱（CAND-005 证据等级 C 的原因）。

## 第七步 · 商业模型 + 防御 -> 插入快筛

- 付费方：买家（成功才付）还是专家（抽佣/担保金池）？未定 = 红旗。
- 防大厂：支付宝担保/猪八戒/淘宝服务市场已有 escrow，但 escrow 对**非标高客单**崩溃（CAND-005 墓碑）。ZJPT 若有壁垒只在"非标结果的可检化"——恰是 CAND-005 证据等级 C、最该成立处尚未被证明的地方。

### 【强制插入 · 项目审问清单 Part A+B 快筛 + 四红线 + 四要素 + 市场基底五问】

四红线（法律/道德/资金5倍/6月无反馈）：
- 6 月无反馈信号：若做"对赌平台本体"——低频 + 供给引擎 + 无飞轮，6 个月内拿到干净复购信号的概率低 🔴（触发红线四倾向）。
- 法律/道德/资金：本体形态无明显单条红线，但"对赌"措辞 + 涉资金结算有合规面（金融/担保牌照风险待查）🟡。

Part A：
- A04 护城河用未来时 🔴（结算护城河已被那个洞证伪，"规模起来就有"= 未来时）。
- A05 通用 AI 可替代 🟡（撮合层易被复制）。
- A03 创始人错配：平台 = 双边市场 + 结算合规，需 BD/合规/支付能力 🟡（与 3-5 人能力画像错配，但按"小团队是起点非天花板"不作否决）。

四要素判据（垂直够小 + 人不可替代 + 动作可接管 + 价值可捕获）：
- 垂直够小：❌"专家对赌平台"是大词，不够小。
- 动作可接管：❌平台本体没把专家的执行动作接管进 AI 内部。
- 价值可捕获：⚠️ 载体性机制——没有"动作可接管"，"价值可捕获"为空（首单后绕单）。
- 结论：四要素在"平台本体"形态下不成立；在"单一窄品类滩头"形态下才可能成立。

市场基底五问：
- 数据类型：撮合/结算数据，平台天然最佳捕捉者 -> 护城河默认归平台不归小团队（同 6-04 电商洞察）。
- 市场轨迹/付费方财务方向：对赌类需求井喷但产物是诉讼；保过类付费方（考生）+ 供方（机构）双承压。
- 官方免费数字化风险 + incumbent lock-in：支付宝/担保/服务市场已占结算-担保层。

-> 快筛结论：**"对赌平台本体"触发 A04🔴 + 四要素两项❌ + 红线四倾向 -> 按 SOP 回第四步换题，不在本体形态继续推。** 保留的是第四步换题后的"CAND-004 滩头形态"。

## 自我攻击（>=3 轮，全部命中同一处）

1. 结算护城河不成立（6-07 那个洞）：高价值·重复·强关系交易，首单建信任=最后一次能干净收费；平台只是供给引擎。
2. "对赌"在中国背剧毒：VAM 诉讼化（1741 起）、中公爆雷（4.82 亿）——承诺不可控结果 = 债务黑洞。
3. CAND-004 使能前提未满足：AI 要把执行搬进供方内部；"专家平台"执行仍在人手里 -> 默认形态踩所有墓碑死因。

## 第八步 · 沉淀 + 资产提取 + 下一步

### 结论（建造>否决，非全盘否）
- 不做：ZJPT "专家对赌结算平台"本体形态（证伪三连 + 快筛触发）。
- 可做：CAND-004 滩头形态——选一个【AI 能端到端执行 + 结果客观快速可测 + 归因干净 + 现按工时/月费卖 + 结果在供方可控范围】的单一窄品类，做"达标否则退费/不收"，供方吃下行。这是把"对结果负责"装进风险代理那条已被验证能活的缝。

### 可迁移资产（待入 case-cards / 铁律候选）
- 资产1：「成功才收费三件套」= 结果客观可测 ∩ 执行在供方手里 ∩ 归因干净（缺一即死）。风险代理活、中公死，差就差在这三件套。
- 资产2：「对赌剧毒律」= 承诺方对结果不可控时，对赌/包退 -> 债务雪球 + 诉讼。评估任何"成功才收费/对赌"先问：结果在承诺方可控范围内吗？
- 资产3：「结算非护城河」复用（来自 6-07）：撮合/结算类平台在高价值低频强关系交易里，护城河不在结算，在持续供给新陌生对——该测复购流向，不是留存。
- 资产4：「分红双关律」= 任何"基础费+分红"代运营，分红必须同时过 归因关（基数只取冻结基线之上的净增量，剔除大盘波动）+ 续费关（分红只绑卖家自己搭不起、且持续变化的供给/系统，绝不绑一次性可学会的运营技巧——否则学得会就断付）。接 CAND-001 + 6-04 持续变化型护城河。

### 下一步（拿包包当协和式 MVP，先测假设，别先设计分红系统）

按"载体死≠假设死，用最便宜 MVP 单独测核心假设"：用包包这一个店手工跑一个月，测三件真东西（按生死排序）：
1. 复购走不走你（生死项）：包包尝到甜头后，是续约分红，还是"我学会了自己来"？= 6-07 那个洞的真实检验。
2. 真专家愿不愿吃下行进来 = A03 创始人错配的真实检验（不是推演）。
3. 净增量能不能归因干净（还是被她自己的供应/定价/大促搅浑）。

分红设计待 MVP 后定，原则先记两关（见下方资产4）：归因关=只分净增量、续费关=分红绑"她搭不起且持续变化的系统"，不绑一次性运营技巧。
