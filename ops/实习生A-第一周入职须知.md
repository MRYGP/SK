## 发布日你的完整流程（每篇文章都这样）

### 发布前（提前2小时）

```
□ 杨发给你文章终稿（md或文档格式）
□ 用编辑器排版（注意：标题、小标题、引用块、加粗、分隔线）
□ 检查：有没有错别字、格式有没有乱、图片有没有缺
□ 预览发给杨确认
□ 杨说OK → 准备发布
```

### 发布（20:30）

```
□ 点发送
□ 确认发布成功（打开文章链接检查一遍）
```

### 发布后 0-5分钟

```
□ 发朋友圈（用杨写好的文案 + 文章链接）
  - 杨的号发一条
  - 你自己的号也发一条（如果愿意）
□ 截图存档（朋友圈发出去的截图）
```

### 发布后 5-15分钟

```
□ 群发到3-5个微信群
  - 每个群用不同的引导语（模板见下方）
  - 不要在所有群发一模一样的话，会显得像广告
```

**群发模板**（直接复制修改）：

AI/创业类群：
> 写了一篇AI产品拆解，发现了一件挺反直觉的事——[用文章里最吸引人的一句话]。5分钟读完，想听听你们的看法。[链接]

朋友/同行群：
> 花了几个月研究的东西，终于写出来了。帮我看看，觉得有道理就帮我点个"推荐"，对新号特别重要。[链接]

行业资源群：
> 最近在研究[文章涉及的行业]方向，写了一篇深度分析。如果你也在关注这个方向，聊聊。[链接]

### 发布后 15-30分钟

```
□ 私信10个种子用户
  - 用杨给的名单，逐一发
  - 模板：[称呼]，写了一篇关于[主题]的拆解，5分钟能读完。
    如果觉得有意思帮我点个"推荐"，对新号冷启动特别重要。[链接]
```

### 发布后 30分钟-2小时

```
□ 评论区运营
  - 杨会写第一条评论（你帮发到评论区）
  - 之后每一条读者评论都要回复
  - 回复风格：真诚、简短、有内容。不要"谢谢关注"这种废话
  - 拿不准怎么回的，截图问杨
```

### 当晚 22:30

```
□ 记录数据到Excel表
  - 阅读量、推荐数、评论数、新关注数
  - 有没有人私信？私信说了什么？（截图存档+转给杨）
```

## 发布后24-48小时要做的事

| 任务 | 怎么做 | 截止 |

| **知乎改编** | 标题改成问答式（比如"为什么XX？"），加强论证段落，去掉文末的招募段落，结尾加一句"更多AI产品拆解，公众号「三湘问道」" | 发布后24小时 |
| **即刻发帖** | 发2-3条短帖（每条不超过200字），角度不同。可以是一个反直觉观点+链接、一组数据+链接、一个提问+链接 | 发布后12小时 |

## 记住一件事

**你的工作核心不是"发文章"，是"让对的人看到文章"。**

发布只是起点。发完之后的2小时冷启动，决定了这篇文章能不能进入算法推荐池。你做的每一个动作——群发、私信、评论区回复——都在帮文章积累初始数据。数据好，算法就会把文章推给更多人。

所以冷启动的那2小时，是你最重要的工作时间。不要拖延，不要漏步骤。
