# AI产品雷达 · 信号源建设指令

> 用途：一次性深度研究，找到全球值得持续跟踪的AI产品信号源（账号/频道/媒体/投资机构）
> 做完这一步后，每周扫描就变成了"看这些人本周发了什么"，而不是"在大海里捞"
> 日期：2026-03-24

---

## 思路

不是"扫平台"，是"跟人"。

好的信号源 = 已经在帮你做筛选的人。他们每天看100个产品，只推荐3个。你跟着他们，等于用了他们的判断力。

所以第一步不是扫描产品，是**扫描信号源本身**。找到那些持续输出高质量AI产品信息的账号/频道/机构。

找到之后，每周扫描就变成了：打开这20个信号源，看他们本周推了什么。15分钟搞定。

---

## 第一步：信号源发现（一次性，用多个AI跑）

### 指令1：抖音/视频号信号源

复制粘贴到AI里：

```
我需要找到中国各平台上，持续介绍、推荐、测评AI应用产品的优质账号。

注意：我要找的不是以下这些类型——
❌ 用AI做内容的账号（比如AI生成视频、AI虚拟人）
❌ AI技术教程/编程教学账号
❌ AI新闻资讯搬运号
❌ AI数字人IP账号

我要找的是这种类型——
✅ 每期介绍1-3个具体的AI产品/工具，告诉你它能干什么、好不好用、适合谁
✅ 有自己的体验和判断，不是只念产品介绍
✅ 会推荐一些小众、冷门但好用的AI产品，不只追热点
✅ 类似"猫不白"（抖音）这种风格——亲自试用后给你推荐

已知参考：猫不白（抖音，质量不错，风格是亲测AI产品然后推荐）

请帮我在以下平台分别找：

1. 抖音：找10个专门做AI工具/产品推荐测评的账号
   搜索建议关键词：AI工具推荐、AI神器、AI产品测评、宝藏AI工具、效率工具推荐

2. B站：找5个专门做AI工具推荐的UP主
   搜索建议关键词：AI工具合辑、AI应用推荐、AI效率工具

3. 小红书：找5个专门做AI产品推荐的账号
   搜索建议关键词：AI神器推荐、AI工具测评、宝藏AI

4. 视频号：找3个做AI产品推荐的账号

5. 即刻：找3个经常分享新AI产品的用户
   搜索建议：AI探索站、出海圈、独立开发者圈

对每个账号，请提供：
| 平台 | 账号名 | 粉丝数 | 更新频率 | 内容风格（一句话） | 最近推荐过哪3个AI产品 | 值得长期跟吗（是/观望/不值得） |

筛选标准：
- 更新频率至少每周1次
- 有自己的使用体验和判断，不是念稿/搬运
- 推荐的产品有具体使用场景
- 优先找粉丝量1万-50万的（太大的往往商业化重，太小的信息不稳定）
```

### 指令2：YouTube/海外信号源

```
我需要找到YouTube和海外平台上，持续介绍AI产品/工具/应用的优质频道和账号。

请帮我找以下类别：

1. YouTube频道（专门做AI产品review/介绍的）
   - 至少找10个
   - 区分：面向开发者的 vs 面向普通用户的 vs 面向创业者的
   - 已知参考：Matt Wolfe (AI tool reviews), The AI Advantage

2. Twitter/X上的AI产品发现者
   - 那些经常第一时间发现和分享新AI产品的人
   - 至少找10个
   - 已知参考：@levelsio, @bentossell (Ben's Bites)

3. Newsletter/邮件订阅
   - 专门做AI产品推荐的newsletter
   - 至少找5个
   - 已知参考：Ben's Bites, The Rundown AI, TLDR AI

4. Reddit/社区
   - 哪些subreddit或社区能最早发现新AI产品？
   - 至少列3个

5. Podcast
   - 专门聊AI产品和AI创业的播客
   - 至少找5个

输出格式：
| 平台 | 账号/频道名 | 订阅数 | 更新频率 | 内容定位 | 推荐质量 | 适合我们跟吗 | 原因 |
```

### 指令3：投资机构/孵化器信号源

```
我需要找到持续投资AI应用层公司的投资机构和孵化器，作为我们发现新AI产品的信号源。

他们投了什么 = 什么方向被聪明钱看好。

请帮我找：

1. 全球AI专项基金/频繁投AI的VC
   - 已知：Y Combinator, a16z, Sequoia, Lightspeed
   - 请补充至少10个，特别是专注AI应用层（不是基础模型）的
   - 每个列出：基金名、重点赛道、近6个月投了哪些AI应用公司、在哪里能看到他们的投资动态

2. 中国投资AI应用的机构
   - 已知：真格基金、红杉中国
   - 请补充至少10个
   - 特别找：投种子轮/天使轮的（这些是最早期的信号）
   - 每个列出：机构名、重点赛道、近6个月投了哪些AI公司、公开信息渠道

3. 孵化器/加速器
   - 已知：YC, Techstars
   - 请补充：中国的AI孵化器/加速器有哪些？
   - 他们最近一期孵化了什么AI项目？

4. 投资数据平台
   - 哪些平台能实时看到AI领域的融资事件？
   - Crunchbase/PitchBook之外，还有什么免费或低成本的替代？
   - 中国的对应平台是什么？（IT桔子？天眼查？）

输出格式：
| 类型 | 名称 | 地域 | AI重点赛道 | 近6月代表案例 | 信息获取渠道 | 跟踪价值（高/中/低） |
```

### 指令4：垂直领域信号源（针对我们的重点方向）

```
我特别关注以下几个AI应用方向。请帮我找到每个方向里，最值得跟踪的信息源（账号/媒体/社区/机构）：

方向1：AI + 银发经济/老年人
- 全球有哪些专门关注银发科技的媒体、研究机构、投资人？
- 中国有没有专注银发赛道的自媒体或社区？

方向2：AI + 内容创作/内容交互
- 谁在持续追踪AI写作、AI视频、AI互动内容领域的产品？

方向3：AI Agent（AI代理）
- 这个方向变化最快，谁的信息最新？哪个社区讨论最活跃？

方向4：AI + 金融/贷款中介
- 有没有专门追踪FinTech AI的媒体或研究机构？

方向5：小团队做大产品（indie hacker/solopreneur）
- 哪些社区/账号专门追踪小团队的AI产品？
- 已知：Indie Hackers, MicroConf

每个方向请至少找3个信号源，说明：名称、平台、更新频率、为什么值得跟。
```

## 第三步：每周扫描（用信号源清单，替代原来的"扫平台"）

信号源清单建好以后，每周扫描指令就简化了：

```
请帮我检查以下信号源在过去7天发布了什么新的AI产品/工具：

[粘贴第一梯队和第二梯队的信号源列表]

对于每个信号源，请告诉我：
1. 本周他们推荐/介绍了哪些AI产品？
2. 哪些产品是我们之前没见过的？
3. 有没有多个信号源同时提到的产品？（交叉验证=强信号）

输出格式：
| 产品名 | 被哪些信号源提到 | 提到次数 | 一句话描述 | 值得深入看吗 |

特别标注：被2个以上信号源同时提到的产品，优先级最高。
```

## 你的思路vs我的思路：怎么融合

| | 你的思路（跟人） | 我之前的思路（扫平台） |
|--|----------------|-------------------|
| 优势 | 精准、高效、利用别人的判断力 | 覆盖面广、能发现冷门信号 |
| 劣势 | 可能有盲区（信号源自己没看到的） | 噪音多、耗时长 |
| 适合 | 主线，每天/每周执行 | 补充，每月一次或特定场景 |

**融合方案：**

日常扫描（每天+每周）= **跟人**（信号源清单）
月度补充（每月一次）= **扫平台**（我之前的通用版指令，捡漏用）
专项扫描（按需）= **银发版 + 死亡版**（这两个保留，因为它们不是"跟人"能覆盖的）

---

## 执行顺序

1. **本周**：用指令1-4跑一遍，建立信号源清单（一次性，2小时）
2. **下周起**：实习生A每天看第一梯队信号源（5分钟），每周看第二梯队（10分钟）
3. **每月**：跑一次通用版扫描指令（补漏）+ 银发版（如果老年AI方向还在推进）
4. **每月**：review信号源清单，淘汰质量下降的，补充新发现的
