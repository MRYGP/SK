```

□ 信号源review：哪个来源产出最多？
```

## 给实习生A的补充

在原来的每日任务之后加15分钟产品扫描。

初筛的感觉：看到一个产品，5个问题每个1秒给答案。"为什么？""怎么可能？""这也行？"——就是有反直觉。"哦，又一个AI工具"——就不是。

第一个月手动做，不用自动化。手动扫建立直觉比工具更重要。
