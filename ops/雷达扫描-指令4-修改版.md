# 雷达扫描 · 指令4修改版（垂直领域信号源）

> 修改原因：千问第一次跑的结果有多处幻觉（虚假星标数）、数据混乱（银发数据串到金融段）、信号源太宏观（推荐咨询报告而非可跟踪账号）
> 本次改进：拆成两次跑、强调要具体可跟踪的账号、排除宏观报告类
> 建议用：Perplexity（首选）或 Claude

---

## 第一次：银发经济 + 金融科技

复制以下内容发给AI：

```
我需要找到"AI+银发经济"和"AI+金融科技"两个方向里，最值得持续跟踪的信息源。

重要要求：
- 我要的是具体的账号/频道/社区/newsletter，不要推荐咨询公司的季度报告（比如麦肯锡报告、普华永道报告这种我不需要，因为每季度才出一次，太慢了）
- 要能每周甚至每天看的
- 每个信号源必须告诉我：名称、平台、更新频率、内容风格、为什么值得跟

方向1：AI + 银发经济/老年人
已知的：AgeClub（中国，微信公众号，质量不错）
请补充：
- 全球有哪些专门关注银发科技/老年科技的媒体、社区、newsletter？
- 特别关注日本、韩国、欧洲的（这些市场老龄化程度高于中国，可能有更多银发AI产品）
- 有没有专门追踪"老年人使用科技产品"的研究机构或媒体？
- 至少找5个信号源

方向2：AI + 金融科技/贷款中介
- 有没有专门追踪FinTech AI产品的媒体、newsletter、社区？
- 有没有专门报道"AI在贷款、信贷、风控领域应用"的账号？
- 中国和海外都要
- 至少找5个信号源

输出格式：
| 方向 | 信号源名称 | 平台 | 语言 | 更新频率 | 内容风格（一句话） | 为什么值得跟 |
```

## 第二次：AI Agent + 内容创作 + 小团队

复制以下内容发给AI：

```
我需要找到"AI Agent""AI内容创作""小团队做大产品"三个方向里，最值得持续跟踪的信息源。

重要要求：
- 要具体的账号/频道/社区，不要推荐CSDN或极客时间这种大平台（太泛了）
- 要那种"跟着他就能第一时间知道这个方向发生了什么"的源头
- 中文和英文都要

方向1：AI Agent
这个方向变化最快。我想知道：
- 谁的信息最新？哪个社区讨论最活跃？
- 有没有专门追踪"新出现的AI Agent产品"的账号/newsletter？
- GitHub上哪些项目/开发者值得watch？
- 至少找5个信号源

方向2：AI + 内容创作/内容交互
- 谁在持续追踪AI写作工具、AI视频工具、AI互动内容领域的新产品？
- 有没有专门做"AI创作工具评测"的账号？
- 至少找3个信号源

方向3：小团队做大产品（indie hacker / solopreneur）
已知的：Indie Hackers社区、MicroConf
请补充：
- 还有哪些社区/账号在追踪"小团队的AI产品"？
- 有没有专门报道"几个人做出了很厉害的AI产品"这种故事的媒体？
- Twitter/X上有没有专门追踪indie AI builders的账号？
- 至少找5个信号源

输出格式：
| 方向 | 信号源名称 | 平台 | 语言 | 更新频率 | 内容风格（一句话） | 为什么值得跟 |
```
