# 高频业务的战略价值

**——为什么美团做酒店能打败携程**

## 一、携程的困惑

2015年，美团宣布进入酒店业务。

携程的CEO在内部会议上说："一个外卖公司做酒店？他们懂什么？"

**2023年，美团酒店间夜量超过携程。**

携程懵了。

**核心问题**：为什么一个"外卖公司"能打败专业的OTA？

**答案**：高频打低频。

---

## 二、核心概念：什么是"高频带低频"

### 定义

- **高频业务** = 用户每天/每周都会用的业务
- **低频业务** = 用户每月/每年才用几次的业务
- **高频带低频** = 用高频业务建立用户心智和习惯，然后以极低边际成本切入低频业务

### 关键机制

1. **心智占领**：用户每天用你，想到相关需求时自然想到你
2. **边际成本趋零**：用户已经在平台上，获客成本为零
3. **数据复用**：高频业务积累的用户画像，可以用于低频业务精准推荐
4. **信任迁移**：高频业务建立的信任，可以迁移到低频业务

## 三、核心案例：美团的"八爪鱼"模型

### 高频业务：外卖（每天）

### 低频业务延伸

| 业务 | 频次 | 与外卖的关系 |
|------|------|-------------|
| 酒店 | 一年几次 | 信任迁移 |
| 机票 | 一年几次 | 信任迁移 |
| 电影票 | 一月几次 | 本地生活心智 |
| 买菜 | 每周 | 配送网络复用 |
| 打车 | 每周 | 用户心智延伸 |
| 医美 | 一年几次 | 本地生活延伸 |

### 数据支撑

- **2023年美团酒店间夜量超过携程**
- 美团打车在部分城市份额超过30%
- 美团买菜成为社区团购头部

### 为什么能成功？

1. **外卖建立了"本地生活找美团"的心智**
2. **用户已经在APP里，切换成本为零**
3. **外卖配送网络可以复用于买菜、跑腿等业务**
4. **用户画像可以精准推荐酒店、餐厅**

### 隐藏武器：地推铁军

- 美团有数万人的地推团队
- 这是携程、滴滴都没有的"重资产"
- 地推能力可以复用于所有本地生活业务

## 五、高频业务的战略价值框架

### 四种战略价值

1. **获客入口**：低成本获取用户
2. **心智占领**：建立"想到X就想到你"的认知
3. **数据积累**：积累用户画像，用于精准推荐
4. **信任迁移**：高频建立的信任，迁移到低频业务

### 成功条件

1. **用户重叠** 2. **场景关联** 3. **能力复用** 4. **信任可迁移**

## 六、对创业者的启示

1. **低频业务**：与高频平台合作，或自建高频入口
2. **高频业务**：思考可延伸的低频业务
3. **中频业务**：警惕被高频降维打击
