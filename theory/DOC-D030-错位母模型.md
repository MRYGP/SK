# DOC-D030 错位母模型

## 一、母模型

**核心结构**:

> **A 应该在 B 的位置,但实际在 C 的位置——这是错位**
> **A 在 B 的位置但时机/标尺/对象错了——这是错配**

错位是空间性的(在哪),错配是关系性的(配不配)。两者经常同时出现。

## 二、错位/错配的五种典型形态

**形态 1:对象错位**——A 服务的对象不是 A 真正能服务的对象
- 例:海外案例平移陷阱(产品对象是中国用户,但形态是为美国用户设计的)
- 例:创始人错配(创始人能力是 A,但产品需要的能力是 B)

**形态 2:阶段错配**——A 处在错的阶段
- 例:DOC-D027 阶段错配
- 例:小团队用大厂打法 / 大团队做小切口

**形态 3:标尺错配**——A 被用错的标尺评估
- 例:种菜 vs 种树标尺(用种树标尺评估种菜方向 = 全否决)
- 例:产品评估用"终局 10 亿"标尺 vs 用"3-6 个月现金流"标尺

**形态 4:语义错位**——A 说的和 B 听的不在同一层
- 例:产品讲"零代码 APaaS",用户搜"不用编程怎么做系统"
- 例:认知翻译层断层
- 例:供给侧叙事 vs 需求侧实际

**形态 5:身份错位**——A 用 B 的姿势做 C 的事
- 例:创业者节奏 vs 投资人节奏 vs 教练节奏(三种身份节奏不同,混着用 = 都做不好)
- 例:用"判断力变现"的姿势做"创业者"的事

## 四、错位识别的元能力

**"看见错位"的认知姿态是稀缺的**。多数人看世界看的是"对称、规律、共性",少数人看的是"不对称、不一致、错位"。

后者是大投资人/创业者的核心能力:
- 索罗斯:认知和现实的反身性错位
- 巴菲特:价格和价值的错位
- 西蒙斯:市场噪音和真实信号的错位
- 贝索斯:基础设施成本和当前定价的错位
- 马斯克:物理极限和工程惯例的错位

**杨的核心能力**:在 AI 应用方向上识别错位。SK 库 90+ 案例本质上都是错位库:产品做的 vs 用户要的、技术能的 vs 商业能的、海外行的 vs 中国能的。

## 五、错位识别的方法论步骤

任何场景下,识别错位的标准动作:

**步骤 1:问"应该是什么样"**
- 用户应该得到什么(对象层)
- 这个阶段应该做什么(时间层)
- 这个事该用什么标尺(标尺层)
- 这件事该谁来做(身份层)

**步骤 2:问"实际是什么样"**
- 实际给了什么 / 在做什么 / 用了什么标尺 / 谁在做

**步骤 3:量化错位距离**
- 距离 = 应然 - 实然
- 距离极大 → 这是机会(可以做产品填补)或风险(踩坑)
- 距离极小 → 没有错位,不是机会

**步骤 4:判断错位的可逆性**
- 错位是结构性的(基础设施级,几年不会变)→ 创业机会
- 错位是周期性的(短期不平衡,自然回归)→ 投机机会,不创业
- 错位是认知性的(只是没人想到)→ 内容/咨询机会

**步骤 5:选择介入位置**
- 在错位的一端介入(供给侧)→ 改造供给
- 在错位的另一端介入(需求侧)→ 教育需求
- 在错位中间介入(翻译层)→ 做认知翻译产品

## 六、为什么这是元理论而非术层

错位/错配模型**不告诉你具体怎么做**,只告诉你"在哪看"——

- 它是其他所有 SKILL 的元判据(每个 SKILL 都在找某种特定错位)
- 它是评估机会的元过滤(没有错位的方向不值得做)
- 它是判断身份的元工具(我现在的身份和动作错位了吗)

**因此放在道层(theory/)而非术层(core/)**,作为 DOC-D030。

## 七、和 SK 库其他元理论的关系

- DOC-D029 客户采用六因子收敛律:**特定形态的错位**(产品和用户采用的错位)
- DOC-D027 阶段错配:**特定形态的错位**(资源和阶段的错位)
- DOC-D028 接盘侠两种性质:**特定形态的错位**(工业化 vs 手工化的错位)
- DOC-D030(本文):**所有错位的母模型**

D029 / D027 / D028 / D023 等都是 D030 的子集。

## 八、使用流程

遇到一个新方向 / 新候选 / 新困境时,先过 5 个错位检查:

1. **对象错位?**——这件事服务的对象,真的是它该服务的对象吗?
2. **阶段错配?**——这件事处在它该处的阶段吗?
3. **标尺错配?**——我在用对的标尺评估它吗?
4. **语义错位?**——我说的和受众理解的在同一层吗?
5. **身份错位?**——我用对的身份姿势做这件事吗?

任一触发 → 暂停往前推进,先消除错位再继续。

## 补丁·2026-05-17:红海是标尺错配(元层应用)

> 来源:杨的观察——"有些热门赛道为什么不能去做呢?也就是说有一些是红海,
> 但也不见得没机会"

### 核心命题

**"红海"本身就是一个标尺错配的产物**。

- 用"竞争者数量"标尺看 → 所有活下来的赛道都是红海
- 用"错位距离"标尺看 → 大多数红海只是第一层被染红,第二层以下还是蓝的

### 红海的真相

用两层断层产品形态扫描:

| 断层 | 红海赛道的状态 |
|---|---|
| 断层一(供给可达性) | ✅ 已填平(红色) |
| 断层二(认知翻译) | ❌ 通常未填平(蓝色) |
| 1.5 层(场景到 SKU) | ❌ 通常未填平(蓝色) |

**红海 = 断层一被填平,断层二/1.5 层仍是空白**。

新品牌的机会不在"卖东西"(断层一),而在:
- "你穿上后的样子"(断层二·认知翻译)
- "这个场景该穿什么"(1.5 层·场景到 SKU)

### 服装行业案例(全五层错位映射)

| 错位类型 | 旧品牌姿势 | 新品牌机会 |
|---|---|---|
| 语义错位 | "纯棉 200 支" | "穿上后的社交信号" |
| 身份错位 | "卖保暖衣" | "身份供应商"(lululemon) |
| 对象错位 | "25-35 岁职场女性" | "副乳困扰运动女性"(MAIA ACTIVE) |
| 阶段错配 | 全球供应链 SKU 海量 | 一个细分场景打透 |
| 标尺错配 | "性价比"标尺 | "情绪舒适"标尺(蕉内) |

### 判别框架:什么时候红海能进?

**三问法**:

1. **断层一填平后,断层二是否仍存在?**
   - 服装:存在 → 能进
   - 充电宝:已填平 → 不能进(除非技术突变)

2. **是否存在语义/身份/对象错位的纠正空间?**
   - 有:用户语言和供给语言之间有翻译层空白
   - 无:用户和供给已完全对齐(标准化工业品)

3. **现有玩家的阶段策略是否固化?**
   - 大品牌用成熟阶段打法 → 小团队用早期阶段打法有机会
   - 行业玩家都在灵活调整 → 红海是真红海

### 反例:什么红海不能进(死海)

用第二波下场判据看:

- **条件 2 不满足**:第一波形态已"足够好"
  - 例:通用搜索引擎(Google 已够好)
- **条件 3 不满足**:第二波位置已被大厂内置功能占住
  - 例:通用电商(淘宝/拼多多已把"场景推荐"内置为算法)

### 一句话

**红海不是颜色的属性,是观察者的标尺。竞争不可怕,
可怕的是你在对手的坐标系里竞争。**
