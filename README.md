## 老年AI管家项目的五条产品准则

1. **零学习** — 用户不需要学。需要"教"的功能，重新设计或不做
2. **习惯>功能** — 养成每天来的习惯，不是做一次性工具
3. **用户自己来** — 不硬拉人，产品好到用户忍不住告诉朋友
4. **功能养出来** — 看数据不看自以为是的设计
5. **在用户已有的地方** — 微信/WhatsApp/LINE，不做独立App
